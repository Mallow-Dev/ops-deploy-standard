# Security

This repository contains operator scripts and demo code. It does NOT contain production secrets.

Responsible disclosure:

- If you find a security issue, please contact the repository owners directly and avoid publishing the vulnerability publicly until a fix is prepared.

Local demo keys:

- Demo private keys are not committed to the repository. Use `./scripts/generate-demo-key.sh` to create a local key in `$XDG_DATA_HOME/ops-deploy-standard`.
